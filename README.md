# Temperature-Gradient-Block-B.Tech-Project
- Designed a Peltier device thermal gradient block using W1209 with a temperature range of 10°C - 50°C across an aluminum plate.
- Utilized temperature gradient system for enhanced multi-temperature cell analysis, optimizing survival and growth dynamics.
